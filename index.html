작동 원리

투명한 설명과 기대치 관리를 위해, 모델의 동작 방식과 한계를 상세히 공개합니다.

1. 업로드

사용자가 선택한 이미지 파일은 브라우저 메모리에서만 처리됩니다. 서버로 전송하거나 저장하지 않습니다.

  • 지원 포맷: JPG, PNG
  • 권장 해상도: 512px 이상
  • 얼굴이 프레임 중앙에 위치

2. 모델 추론

Teachable Machine 기반 이미지 분류 모델을 사용해 강아지상/고양이상 확률을 계산합니다.

  • 확률 기반 결과 제공
  • 예측값은 절대적 진단이 아님
  • 광원·각도·표정에 민감

3. 결과 표시

확률 값을 시각적으로 표시하고, 최상위 클래스를 강조합니다. 모든 결과는 참고용입니다.

  • 실시간 막대 그래프
  • 즉시 확인 가능
  • 결과 저장 기능 없음

4. 책임 있는 사용

이 서비스는 오락 목적이며, 외모 평가나 차별적 용도로 사용하지 않습니다.

  • 민감 정보 판단 금지
  • 연령·성별 추정 기능 없음
  • 유해 콘텐츠 업로드 금지

고품질 콘텐츠 & 신뢰 기준

사용자 경험을 우선으로, 명확한 설명과 투명한 운영 원칙을 제공합니다.

서비스 소개

동물상 테스트는 사진 기반으로 강아지상/고양이상 유사도를 분석하는 데모 서비스입니다. 흥미와 재미를 위한 참고용 결과를 제공합니다.

  • 사진은 브라우저에서만 처리됩니다.
  • 모델은 공개된 Teachable Machine 기반입니다.
  • 결과는 의료·전문 판단이 아닙니다.

콘텐츠 품질

동일한 내용의 반복이 아닌, 서비스 목적과 사용 가이드를 충분히 설명해 정보 가치를 높입니다.

  • 기능·제한·주의사항을 명확히 안내
  • 사용자에게 기대치를 분명히 전달
  • 모호한 과장 표현을 지양

사용 가이드

정면 얼굴이 잘 보이는 사진이 예측 정확도를 높입니다. 조명, 해상도, 각도에 따라 결과가 달라질 수 있습니다.

  • 밝은 자연광 환경 권장
  • 얼굴이 프레임 중앙에 위치
  • 지나친 보정 사진은 비권장

신뢰와 안전

사용자의 신뢰를 우선으로 운영합니다. 광고보다 콘텐츠가 우선이며, 과도한 광고 배치를 지양합니다.

  • 사용자 경험을 해치지 않는 배치
  • 정확한 정보 제공을 우선
  • 필요 시 즉시 개선·수정

업로드한 사진이 저장되나요?

아니요. 이미지는 브라우저에서만 처리되며 서버로 전송되거나 저장되지 않습니다.

결과가 바뀌는 이유는 무엇인가요?

조명, 각도, 표정, 해상도에 따라 모델이 인식하는 특징이 달라질 수 있습니다.

모델은 어떻게 학습되었나요?

공개된 Teachable Machine 도구를 기반으로 이미지 분류 모델을 구성했습니다.

광고가 표시되나요?
